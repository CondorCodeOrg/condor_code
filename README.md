# CondorCode

An open-source, gamified learning platform for developers.

## 🎯 What is CondorCode?

CondorCode is a joyful, beautiful, and fun learning experience platform built by **Condor Coders**!

We focus on **cross-platform development** (Dart & Flutter) but it's not the end goal!

Right now, we have Ukrainian language course content for Dart and Flutter beginners.

<video src="assets/demo.mp4" width="640" controls muted loop playsinline></video>

### What we want to build

- Follow structured **roadmaps** to master a skill from zero to hero
- Solve **quizzes**, **coding tasks**, and **challenges**
- Earn **rewards** and track progress like in a game
- Communicate with other learners and share progress
- Read curated **blog** posts and useful information
- Enjoy a **personalized experience** adapted to their level

### What contributors can do

Anyone can contribute to make this platform better:

- Suggest and add new features
- Improve UI/UX and accessibility
- Fix bugs and optimize performance
- Write documentation and tutorials

---

## 🛠️ Tech Stack

- **Flutter** — cross-platform UI (mobile + web)
- **Firebase** — backend, authentication, analytics, hosting
- **Melos** — monorepo tooling
- **Clean Architecture** — scalable, maintainable codebase
